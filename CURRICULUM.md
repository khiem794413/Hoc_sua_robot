# Lộ Trình Học Sửa Robot

Mục tiêu: Từng bước từ kiến thức cơ bản đến chuyên sâu. Mỗi module gồm bài học (theory), lab (practical), quiz/checklist và project nhỏ.

Giai đoạn A — Cơ bản (0 → 3 tháng)
- Module 01: Điện tử & Điện cơ cơ bản
- Module 02: Lập trình cơ bản (Python, C) & Git
- Module 03: Toán & Kỹ thuật điều khiển (số học, ma trận, PID cơ bản)

Giai đoạn B — Trung cấp (3 → 9 tháng)
- Module 04: Cảm biến & Actuator (encoder, IMU, lidar, camera)
- Module 05: Robot kinematics & dynamics
- Module 06: ROS2 cơ bản (nodes, topics, services, actions)

Giai đoạn C — Nâng cao (9 → 18 tháng)
- Module 07: Tự động định vị & lập bản đồ (SLAM)
- Module 08: Thị giác máy tính & deep learning cho robot
- Module 09: Tối ưu hoá & an toàn chức năng

Giai đoạn D — Chuyên gia
- Module 10: Thiết kế hệ thống điều khiển fault-tolerant
- Module 11: Sửa chữa nâng cao trên robot công nghiệp thực tế
- Capstone Project: Nhóm thực hiện khắc phục/thiết kế hệ robot thực tế (tài liệu, video, test)

Mỗi module gồm:
- README (mục tiêu, yêu cầu đầu vào)
- Lessons (theory)
- Labs (mô tả bài tập + checklist)
- Quiz/Assessment
- Ví dụ mã & scripts mô phỏng
