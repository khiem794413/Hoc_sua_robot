# Học Sửa Robot — Từ Không Biết Đến Bậc Thầy

Mục tiêu: Cung cấp một lộ trình học tập thực hành, có cấu trúc, để bất kỳ ai — từ người mới bắt đầu — trở thành chuyên gia chẩn đoán, sửa chữa và bảo trì robot công nghiệp, robot dịch vụ và robot tự động.

Nội dung chính:
- Lộ trình học tập theo mô-đun (từ cơ bản đến nâng cao)
- Bài tập thực hành, bài lab, danh sách công cụ
- Dự án cuối khoá (capstone) để chứng minh năng lực
- Tài liệu, sách và khoá học tham khảo

Bắt đầu nhanh:
1. Clone repository:
   git clone https://github.com/khiem794413/Hoc_sua_robot.git
2. Mở file CURRICULUM.md để xem lộ trình chi tiết.
3. Chọn một module, làm theo các bài học và hoàn thành lab.

Đóng góp:
- Mở issue cho lỗi/tài nguyên bị thiếu.
- Gửi Pull Request để thêm bài lab, sửa lỗi chính tả hoặc cập nhật tài nguyên.
